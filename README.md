# Mollick Software Solutions Website

Premium React + Vite website for Mollick Software Solutions.

## Run in VS Code

1. Open this folder in VS Code.
2. Open the terminal.
3. Install dependencies:

```bash
npm install
```

4. Start the website:

```bash
npm run dev
```

5. Open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
